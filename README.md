# Neighborhood Map React

This is a project for Front-End Web Developer Udacity Nanodegree Program.

### Usage

You need Node.js installed on your computer to run the app

To run app:

Clone the repository
Execute: npm install
Execute: npm start

To run with ServiceWork
Execute: npm run build


### External API

Power by :

Google Map API - https://developers.google.com/maps/documentation/
FourSquare API - https://developer.foursquare.com/docs/api/
